#General info

run
```
npm install
```

or run
```
npm i -D selenium-webdriver
npm i -D puppeteer
npm i -D cypress
npm i -D playwright
npm i -D @playwright/test
```

run with node:
```
node _test.js
node test_selenium.js
node test_puppeteer.js
node test_playwright.js
```

run with playwright/test:
```
npx playwright test
```
